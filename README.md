# URDF_ASSEST
本仓库集成市面上常见的URDF模型，集成为资产库

## 工具使用

### 生成 manifest.json
本项目提供了一个用来一键为机器人模型文件夹生成 `manifest.json` 描述文件的脚本：

```bash
python utils/manifest.py <机器人模型文件夹路径>
```

**功能说明：**
- 它可以自动扫描该文件夹下的图片、urdf、mjcf 和相应的网格数据（stl / obj / dae）。
- 运行过程中会依次引导你设置或选择机器人名称、描述、预览图和对应文件。
- 对于所有的文件，你可以直接敲击回车使用默认选项，或者是输入列表对应的数字选择其他文件。
